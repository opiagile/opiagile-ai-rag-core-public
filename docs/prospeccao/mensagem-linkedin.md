# Mensagem Para LinkedIn

```text
Tenho um core RAG técnico da Opiagile para produtos de atendimento com IA: upload de documentos, respostas com fontes, embeddings/pgvector quando configurados, LLM real validada, memória conversacional, triagem de intenção, qualificação de leads, handoff humano, observabilidade e contrato para frontend/gateways separados.

O projeto usa Java/Spring Boot, PostgreSQL/pgvector, Docker Compose e modo local sem chave externa. Posso adaptar esse core para os documentos e fluxos do seu negócio, e integrar canais como WhatsApp ou Teams em gateways separados.
```

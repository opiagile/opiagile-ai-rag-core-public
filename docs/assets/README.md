# Assets Visuais

Esta pasta fica reservada para evidências visuais de projetos consumidores do core, como um futuro `opiagile-rag-demo-web`.

O frontend foi removido deste repositório. Para criar screenshots ou GIFs, implemente primeiro a interface separada usando [`../frontend-handoff.md`](../frontend-handoff.md).

## Capturas Recomendadas Para O Projeto Frontend

Capturar manualmente, sem dados sensíveis:

1. status da API;
2. upload de `samples/clinica/faq.txt`;
3. pergunta com resposta e fontes;
4. handoff e trace;
5. versão responsiva mobile.

## Cuidados

- Não capturar `.env`, tokens, chaves, senhas ou dados reais.
- Usar apenas dados fictícios ou previamente aprovados.
- Evitar caminhos locais desnecessários na imagem.
- Conferir visualmente cada imagem antes de commitar.
